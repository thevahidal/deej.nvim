INTRODUCTION                                    *deej*

Deej.nvim is a Neovim plugin that plays rhythmic beats as you type code, turning
your coding session into a DJ-like experience, inspired by Teej. Each keypress
triggers a sound, with different beats for regular characters, enter, braces, or
semicolons. Supports themes for different sound profiles and an optional
background loop.

INSTALLATION                                    *deej-installation*

1. Install using your favorite plugin manager:

   With `lazy.nvim`:
   >
   { 'thevahidal/deej.nvim', config = function() require('deej').setup() end }
   <

   With `packer.nvim`:
   >
   use 'thevahidal/deej.nvim'
   <

   With `vim-plug`:
   >
   Plug 'thevahidal/deej.nvim'
   <

2. Install an audio player:
   - Linux: `sudo apt-get install alsa-utils mpv`
   - macOS: `brew install mpv`
   - Windows: Install `mpv` and add to PATH, or use WSL.

3. Download WAV beat samples (e.g., kick.wav, snare.wav) and optionally a loop
   track (e.g., background_loop.wav) for each theme. Place them in
   `~/.local/share/nvim/site/data/deej/beats/`.

CONFIGURATION                                   *deej-configuration*

Add to your `init.lua`:
>
require('deej').setup({
  beat_dir = vim.fn.stdpath('data') .. '/deej/beats/',
  themes = {
    default = {
      beat_files = {
        default = 'kick.wav',
        enter = 'snare.wav',
        brace = 'hihat.wav',
        semicolon = 'clap.wav',
      },
      loop = nil,
    },
    techno = {
      beat_files = {
        default = 'techno_kick.wav',
        enter = 'techno_snare.wav',
        brace = 'techno_hihat.wav',
        semicolon = 'techno_clap.wav',
      },
      loop = 'techno_loop.wav',
    },
  },
  active_theme = 'default',
  cooldown = 0.1, -- Time between triggered sounds (seconds)
  volume = 50,    -- Volume for triggered beats (0-100)
  loop_volume = 30, -- Volume for loop track (0-100)
})
<

COMMANDS                                        *deej-commands*

*DeejToggle*  Toggle the plugin and background loop on/off.
>
:DeejToggle
<

*DeejSetTheme*  Switch to a different sound theme (e.g., `:DeejSetTheme techno`).
>
:DeejSetTheme <theme_name>
<

MAPPINGS                                        *deej-mappings*

Suggested mappings:
>
vim.keymap.set('n', '<leader>dj', require('deej').toggle, { desc = 'Toggle Deej' })
vim.keymap.set('n', '<leader>dt', ':DeejSetTheme ', { desc = 'Set Deej Theme' })
<

USAGE                                           *deej-usage*

- Type code to trigger beats based on the active theme.
- A background loop plays if configured in the theme (requires mpv).
- Use `<leader>dj` or `:DeejToggle` to toggle the plugin and loop.
- Switch themes with `:DeejSetTheme <theme_name>`.
- Customize themes by defining new ones or replacing sound files.
